### 五、项目介绍
项目编号：mpweixinA237D

基于微信小程序的公司点餐系统，为企业提供便捷的订餐服务，实现菜品展示、在线点餐、订单管理等功能，方便职工就餐，提升餐饮管理效率。

角色：管理员、用户、商家

用户功能：注册登录、浏览菜品、菜品分类、在线点餐、订单管理、评价评论、收藏菜品、个人中心。

商家功能：登录、菜品发布、菜品管理、订单处理、评价查看、销售统计。

管理员功能：登录、用户管理、商家管理、菜品管理、订单管理、菜品分类管理、评论管理、系统配置。

数据库表：共17张表
